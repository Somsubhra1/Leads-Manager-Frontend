# LeadsManager Frontend

## Description

Frontend for a Leads Manager App using ReactJS, Redux & Axios

## Installation

```sh
npm install
```

## Running the app

```sh
$ npm start
```

## Build the app

```sh
$ npm run build
```

## Setup

Create .env file in the root directory and add the following configuations:

```
REACT_APP_API_BASE_URL - Set API base URL
```
